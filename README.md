# Customer Churn Analysis and Prediction

## Project Overview

This project analyzes customer data and predicts customer churn using Machine Learning. The goal is to identify customers who are likely to leave a telecom service and help businesses improve customer retention.

## Objective

To analyze customer behavior and predict customer churn using Logistic Regression.

## Tools Used

* Python
* Google Colab
* Pandas
* Matplotlib
* Seaborn
* Scikit-Learn

## Dataset Features

* CustomerID
* Gender
* Tenure
* MonthlyCharges
* Contract
* Churn

## Steps Performed

1. Data Collection
2. Data Loading and Exploration
3. Data Visualization
4. Data Preprocessing
5. Train-Test Split
6. Logistic Regression Model
7. Model Evaluation
8. Confusion Matrix Analysis

## Results

* Successfully predicted customer churn.
* Generated visualizations for customer analysis.
* Evaluated model performance using accuracy score and confusion matrix.

## Conclusion

The Logistic Regression model was successfully trained and tested. The project helps identify customers who are likely to leave and supports customer retention strategies.
